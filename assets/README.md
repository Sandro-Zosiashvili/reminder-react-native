# Assets Directory

This directory should contain the following image files for the Expo app:

## Required Assets

1. **icon.png** (1024x1024)
   - App icon shown on device home screen
   
2. **splash.png** (1284x2778 recommended)
   - Splash screen shown when app launches
   
3. **adaptive-icon.png** (Android only, 1024x1024)
   - Adaptive icon for Android devices
   
4. **notification-icon.png** (96x96)
   - Icon shown in notifications
   
5. **favicon.png** (48x48)
   - Favicon for web version

## Temporary Setup

For development, you can create simple placeholder images or use default Expo assets.

To generate these assets quickly:
1. Use a tool like Figma or Canva
2. Use the app icon generator: https://www.appicon.co/
3. Or simply create solid color PNGs with the required dimensions

## Color Scheme for Assets

Based on the app's primary colors:
- Primary: #5B63D3 (Blue-Purple)
- Background: #0A0E27 (Dark Navy)
- Success: #00D9A5 (Teal)

## Note

The app will run without these assets in development mode, but they're required for production builds.
